# SwapIt
A localized marketplace for college students to trade and buy campus essentials.

## Tech Stack
- **Framework:** React Native (Expo)
- **Navigation:** Expo Router
- **Database:** Supabase
- **Data Fetching:** TanStack Query
- **Maps:** react-native-maps
